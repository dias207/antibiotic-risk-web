# Антибиотикке төзімділік тәуекелдерін тексеру қосымшасы

Әртүрлі елдерде тері микрофлорасының антибиотиктерге төзімділігінің деңгейін бағалауға арналған веб-қосымша.

## Функционал

- ✅ Адам нөмірін енгізу және саяхат ететін елді таңдау
- ✅ Тәуекел деңгейін көрсету (қызыл/сары/жасыл үшбұрыш)
- ✅ Ең қауіпті бактериялар туралы ақпаратпен геймификация
- ✅ Қызыл аймақ үшін тұру мерзімі туралы қосымша сұрақ
- ✅ Геолокация және елге оралғанда хабарландырулар
- ✅ Еске салулармен ақылды push-хабарландырулар

## Пайдалану

1. **Браузерде ашу:**
   - `index.html` файлын екі рет басып ашыңыз
   - Немесе файлды браузерге тартып тастаңыз
   - Немесе браузерде `Ctrl+O` басып файлды таңдаңыз

2. **Қосымшаны пайдалану:**
   - Адам нөмірін енгізіңіз
   - Саяхат ететін елді таңдаңыз
   - "Тәуекелді тексеру" батырмасын басыңыз
   - Қажет болса, геолокация байқауын қосыңыз

## Технологиялар

- HTML5
- CSS3 (анимациялар мен градиенттермен)
- Vanilla JavaScript
- LocalStorage (күй сақтау үшін)
- Geolocation API
- Notifications API

## Ерекшеліктер

- Бэкендсіз толығымен жұмыс істейді
- Адаптивті дизайн мобильді құрылғылар үшін
- Заманауи UI анимациялармен
- Әр тәуекел аймағы үшін бактериялар туралы ақпарат
- Дәрігерлер үшін сандық ізді тексеру режимі

## Ескертулер

- Геолокация жұмысы үшін пайдаланушы рұқсаты қажет
- Push-хабарландырулар браузер рұқсатын талап етеді
- Елдер мен бактериялар туралы деректер кодта жергілікті сақталады
